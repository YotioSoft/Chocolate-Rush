				ChocolateRush

------------------------------------------------------------------------------

1.はじめに
　この度は、ChocolateRushをダウンロードしていただき、ありがとうございます。
　このゲームは、空から降ってくるチョコレートを
　画面下部に表示される男の子を操作して受け取るゲームです。

2.操作方法
　ChocolateRush.exeを起動してください。
　ゲーム中はBGMが割りと大きめな音で流れますので、音量にご注意ください。
　
　←　→：男の子を左右に移動させます。

3.ルール
　受け取った品物によって加点される点数が異なります。
　・「LOVE」チョコ　　　　：3点
　・板チョコレート（3種） ：2点
　・フレンチクルーラー　　：2点
　・義理チョコ　　　　　　：1点
　
　また、下記の品物（異物）を受け取ると2点ずつ減点されます。
　・ダンボール
　・琴
　・USBメモリ
　・長財布
